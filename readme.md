# Tk GUI builder

The purpose of this project is to create a GUI design tool for Python Tkinter. It mainly uses pygame and tkinter.

## Getting started

This project uses Python 3.14 ([python.org](python.org)) and libraries mentioned in requirements.txt.

	pip install -r requirements.txt

It is recomennded to create an virtual environment.

	python3 -m venv env
	source env/bin/activate
	env/bin/python -m pip install -r requirements.txt
	deactivate

	chmod 755 ./main.py
	env/bin/python ./main.py
